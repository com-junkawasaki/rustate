# Rustate Agent (プロトタイプ)

Rustate Agent は、Rust で実装された状態機械駆動の LLM エージェントフレームワークの**プロトタイプ**です。このライブラリは [statelyai/agent](https://github.com/statelyai/agent) を参考に開発されており、Rust の型システムと所有権モデルを活かした安全で効率的な実装を目指しています。

## 現在の実装状況と特徴

- **状態機械駆動**: `rustate` を使用した明示的な状態遷移（現在はシンプルな Todo リスト管理を実装）
- **LLM エージェント連携**: OpenAI (GPT-3.5-turbo) と連携し、状態、コンテキスト、履歴、フィードバックに基づいて次のアクションを決定
- **観測とフィードバック**: エージェントの行動（観測）と、その結果に対するフィードバックを記録する基本的な仕組み
- **非同期サポート**: `tokio` を使用した非同期処理
- **基本的な CLI**: `add`, `view`, `goal` コマンドによる対話的操作と、Idle 状態でのエージェントの自律動作

## 主要コンセプト (目標)

このプロジェクトは以下のコンセプトの実現を目指しています（一部は実装途上または未実装）:

- **観測 (Observations)**: 状態遷移の記録（実装済み）
- **決定 (Decisions)**: エージェントが選択するアクション（実装済み、改善の余地あり）
- **フィードバック (Feedback)**: 決定に対する評価や報酬（実装済み）
- **洞察 (Insights)**: 状態遷移に関する追加コンテキスト（未実装）
- **エピソード (Episodes)**: 初期状態から目標状態までの完全な状態遷移シーケンス（未実装）
- **共有コンテキスト (Shared Context)**: 複数のエージェント間で共有されるデータ（構想段階）
- **共有状態機械 (Shared Machines)**: 複数のエージェントから操作可能な状態機械（構想段階）
